Safe Start

安全に始める手順

いきなり家庭全体へ適用せず、影響範囲を小さく保ちながら段階的に確認します。

Portal に戻る
1

起動確認

Portal、Pi-hole、NetAlertX、Uptime Kuma の画面が開けることを確認します。この段階ではルーターや端末の DNS 設定は変更しません。

2

ローカル監視

Uptime Kuma でローカルサービスだけを監視します。NetAlertX は自宅 LAN のみを対象にし、通知連携は必要になってから設定します。

3

一台だけ Pi-hole を試す

Windows PC など自分の端末だけ DNS を Raspberry Pi の IP へ向けます。問題が出たら DNS 設定を自動取得へ戻します。

禁止事項

  • 53 番ポートをインターネットへ公開しない。
  • 管理画面をポート開放しない。
  • 大学、研究室、会社、共有 Wi-Fi で無断スキャンしない。
  • .env、data、backups を GitHub へ公開しない。
